Create File Header

Although the GEOTRANS coordinate file format is very simple, creating the header for an input coordinate file manually can be an error prone process. To automate this, you can use the Create File Header command, which is located in the File menu, as follows:

1.   Select the Create File Header command from the File menu.  The Create File Header window will appear.

2.   In the Create File Header window, specify the datum, the coordinate reference system type, and any associated parameters for the input coordinate file that you desire.  Then click the Create button, which will cause a standard file Save As dialog box to appear.

3.   In the Save As dialog box, specify a file name and directory for the resulting coordinate file header.  Click on the Save button to cause the file to be created, and return to the GEOTRANS main window.

The resulting file will contain the specified header information.  This header can then be attached to an existing text file of coordinates to create a GEOTRANS input coordinate file.
